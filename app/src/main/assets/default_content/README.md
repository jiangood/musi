# Literal Musi

Listen. Search. Move on.

A minimalist music player with Git sync.

## Features

- Audio Files: Play mp3, flac, ogg, opus, wav, m4a, aac
- Git Sync: Sync your library across devices via your own GitHub repo
- Search: Find any track instantly by file name
- Minimal: No playlists, no metadata, no algorithm

## Deletion

When you delete a track:

1. It moves from pile/ to trash/ in your remote repository
2. It disappears from the app immediately
3. There is no local trash — deletion syncs directly

To permanently delete: remove files from trash/ in your repo.

There is no restore button. No undo. This is intentional.

## Philosophy

Literal Musi is not about organizing your music.

It's about not needing to.
